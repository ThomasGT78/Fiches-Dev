Lorsque vous cliquez sur le bouton « Envoyer », le formulaire vous amène alors à la page indiquée dans l'attribut action On peut changer le texte affiché à l'intérieur des boutons avec l'attribut value (BOUTON D'ENVOI) Bouton d'envoi de formulaire. Le visiteur sera conduit à la page indiquée dans l'attribut action du formulaire (BOUTON D'ENVOI GRAPHIQUE) Équivalent du bouton submit, sous forme d'image. Rajoutez l'attribut src pour indiquer l'URL de l'image (RESET) Remise à zéro du formulaire (BOUTTON MULTI USAGE) Bouton générique, qui n'aura par défaut aucun effet. Souvent géré en JavaScript pour exécuter des actions sur la page. on peut faire que n'importe quel input Reset ou Submit un formulaire.
